TableColumn represents a column in a table, identified by its index and consisting of a list of table cells.
You can access a specific cell within the column using its index with
getCell()
.
Methods are available to retrieve the column's index, the number of cells it contains, and its parent table.
You can also get the width of the column in points using
getWidth()
.
The column can be removed using the
remove()
method, which may also remove the table or other columns if cells are merged.

remove()
Removes the table column.
If all the cells in the column are merged with other columns, the common columns spanned by
these cells are removed.
If no columns remain in the table after this removal, the whole table is removed.
