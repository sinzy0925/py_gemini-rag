SheetsChart represents a linked chart embedded from Google Sheets.
You can manipulate the chart's position, size, and rotation on the page.
The chart can be refreshed to get the latest data from the source Google Sheet.
You can get the ID of the source spreadsheet and the specific chart within it.
It is possible to set or remove links on the SheetsChart element.

that page element; it doesn't affect the transforms of the group or other page elements in the
group.
