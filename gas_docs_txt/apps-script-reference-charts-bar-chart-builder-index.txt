BarChartBuilder is a builder for creating bar charts, with more details available in the Google Charts documentation.
You can build a bar chart by setting properties like title, axis titles, dimensions, legend position, and data source URL.
Data for the bar chart can be imported from external sources like Google spreadsheets using
setDataSourceUrl
.
The BarChartBuilder offers various methods to customize the chart's appearance and behavior, including setting colors, background color, text styles, and stacked options.
The
build()
method returns a Chart object which can be used for embedding or as a static image.

reverse
Categories()
Reverses the drawing of series in the domain axis. For vertical-range charts (such as line,
area or column charts), this means the horizontal axis is drawn from right to left. For
horizontal-range charts (such as bar charts), this means the vertical axis is drawn from top to
bottom. For pie charts, this means the slices are drawn counterclockwise.

grow from left to right. Calling this method causes them to grow from right to left.
