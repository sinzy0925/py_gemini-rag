ColumnChartBuilder is a builder for creating column charts.
You can set data using a DataTable or a data source URL.
The builder provides methods to customize chart appearance, including title, axis titles, dimensions, colors, and background.
You can set advanced options using the
setOption
method.
The
build()
method returns a Chart object that can be used or embedded.

reverse
Categories()
Reverses the drawing of series in the domain axis. For vertical-range charts (such as line,
area or column charts), this means the horizontal axis is drawn from right to left. For
horizontal-range charts (such as bar charts), this means the vertical axis is drawn from top to
bottom. For pie charts, this means the slices are drawn counterclockwise.
