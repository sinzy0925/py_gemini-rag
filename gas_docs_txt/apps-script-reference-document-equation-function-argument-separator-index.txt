The EquationFunctionArgumentSeparator represents a function separator in a mathematical Equation element and cannot contain other elements.
You can copy the EquationFunctionArgumentSeparator element, retrieving a detached, deep copy of the current element.
You can retrieve the element's attributes, next sibling, parent, previous sibling, and type.
You can check if the element is at the end of the Document.
You can merge the element with a preceding sibling of the same type or remove it from its parent.
You can set the element's attributes.
