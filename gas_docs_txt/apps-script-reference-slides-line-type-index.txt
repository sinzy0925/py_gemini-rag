LineType represents various line types derived from a subset of ST_ShapeType values in ECMA-376.
To use an enum, you call its parent class, name, and property, such as
SlidesApp.LineType.STRAIGHT_CONNECTOR_1
.
The available LineType properties include different straight, bent, and curved connectors, as well as a straight line and an unsupported type.
Line
Type
The line types.
Derived from a subset of the values of the "ST_ShapeType" simple type in section 20.1.10.55 of
"Office Open XML File Formats - Fundamentals and Markup Language Reference", part 1 of

edition
