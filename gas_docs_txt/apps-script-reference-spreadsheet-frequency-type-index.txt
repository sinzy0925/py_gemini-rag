Frequency
Type
An enumeration of frequency types.
To call an enum, you call its parent class, name, and property. For example,
SpreadsheetApp.FrequencyType.DAILY
.
Properties
Property
Type
Description
FREQUENCY_TYPE_UNSUPPORTED

The frequency type is unsupported.

Refresh daily.
WEEKLY

MONTHLY
