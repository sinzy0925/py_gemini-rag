AggregationType is an enum in Apps Script that defines aggregation types for a Field.
You access enum values through its parent class, name, and property, like
DataStudioApp.AggregationType.AVG
.
Available aggregation types include AVG, COUNT, COUNT_DISTINCT, MAX, MIN, SUM, and AUTO.
The
AUTO
property is used for calculated fields referencing an aggregated field.
The
NO_AGGREGATION
property is deprecated and should be replaced with
AUTO
.
