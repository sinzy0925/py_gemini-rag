ChatResponse is the response object for a card message in Google Chat, available only for Google Chat apps.
You can create a ChatResponse using
CardService.newChatResponseBuilder()
, adding text and cards using
setText()
and
addCardsV2()
.
The
printJson()
method is available for debugging and prints the JSON representation of the ChatResponse object.
