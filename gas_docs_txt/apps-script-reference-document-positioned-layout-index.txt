Positioned
Layout
An enumeration that specifies how to lay out a
Positioned
Image
in relation to surrounding
text.
To call an enum, you call its parent class, name, and property. For example,
DocumentApp.PositionedLayout.ABOVE_TEXT
.
Properties
Property
Type
Description
ABOVE_TEXT
Enum
The image is positioned above the text.
BREAK_BOTH
Enum
The image breaks the text on the left and right.
BREAK_LEFT
Enum
The image breaks the text on the left.
BREAK_RIGHT
Enum
The image breaks the text on the right.
WRAP_TEXT
Enum
The image is wrapped by text.
