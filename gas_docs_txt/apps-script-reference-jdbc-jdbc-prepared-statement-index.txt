The
JdbcPreparedStatement
class in Apps Script facilitates interaction with SQL databases using prepared statements, mirroring Java JDBC API functionality.
Key methods are available for batch operations, statement execution, setting parameters, retrieving information, and managing the statement itself.
Numerous
set
methods are provided to configure parameters for the prepared statement at specific indices, supporting various data types.
Statement properties can be configured using methods like
setCursorName
,
setFetchDirection
,
setMaxRows
, and
setQueryTimeout
.
Most methods require authorization with the
https://www.googleapis.com/auth/script.external_request
scope.
